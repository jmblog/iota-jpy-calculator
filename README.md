# iota-jpy-calculator

## Installation

### Prerequisites

Install [polymer-cli](https://github.com/Polymer/polymer-cli) globally.

```bash
npm install -g polymer-cli
```

### Setup

```bash
$ npm install
```

## Development

### Start the development server

```bash
$ npm run start
```

### Run web-component-tester tests

```bash
$ npm run test
```

### Build

```bash
$ npm run build
```

### Test the build

```bash
$ npm run test-build
```
